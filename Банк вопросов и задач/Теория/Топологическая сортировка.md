# Топологическая сортировка

## Вопрос

Объясните тему «Топологическая сортировка»: как устроен механизм, какие инварианты определяют поведение и где проходят практические границы?

## Короткий ориентир

Вопрос заметки: как построить порядок зависимостей и одновременно доказать, что такой порядок вообще существует?

Topological order directed graph — перестановка vertices, в которой для каждого edge `u→v` вершина `u` стоит раньше `v`. Такой порядок существует тогда и только тогда, когда graph — directed acyclic graph (DAG).

Алгоритм Kahn последовательно удаляет vertices с текущей indegree `0`. Если обработано меньше `V` вершин, оставшийся подграф содержит directed cycle. DFS даёт альтернативу: reverse finishing order корректен, только если во время обхода не найден back edge в active vertex.

Оба варианта работают за `Θ(V+E)` на adjacency lists. Конкретный порядок не обязан быть единственным; если нужен детерминированный лексикографически минимальный результат, zero-indegree set хранят в priority queue и платят логарифмический множитель.
