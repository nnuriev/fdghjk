# Платформа

## Назначение

Прикладные Go-задачи, snippets и code review, где результат зависит от семантики языка, runtime, memory model, стандартной библиотеки или concurrency primitives.

## Входные знания

Перед практикой полезно пройти [[Банк вопросов и задач/Теория/Карта — Теория|теоретические вопросы]] и [[60 Go/Карта — Go|карту Go]].

## Маршрут

Сначала разбирайте контракт и наблюдаемый результат, затем объясняйте механизм и только после этого исправляйте код.
