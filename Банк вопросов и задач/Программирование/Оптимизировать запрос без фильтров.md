# Оптимизировать запрос без фильтров

## TL;DR

В материалах Avito сохранилось только название «Оптимизировать запрос без фильтров». Этого недостаточно, чтобы восстановить алгоритм, SQL или ожидаемый результат без догадок.

## Наблюдаемая формулировка

Единственная подтверждённая формулировка: **«Оптимизировать запрос без фильтров»**. Она перечислена в [[Авито/roadmap#DB и аналитика|разделе «DB и аналитика»]] среди заданий без сохранённого условия.

## Что отсутствует в контракте

- полный SQL-запрос и схема;
- объёмы и распределение данных;
- execution plan и целевая метрика.

Карточка не объединяется с похожими задачами, пока основной контракт и ожидаемый результат неизвестны.
