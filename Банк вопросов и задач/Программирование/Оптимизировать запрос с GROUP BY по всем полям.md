# Оптимизировать запрос с `GROUP BY` по всем полям

## TL;DR

В материалах Avito сохранилось только название «Оптимизировать запрос с GROUP BY по всем полям». Этого недостаточно, чтобы восстановить алгоритм, SQL или ожидаемый результат без догадок.

## Наблюдаемая формулировка

Единственная подтверждённая формулировка: **«Оптимизировать запрос с `GROUP BY` по всем полям»**. Она перечислена в [[Авито/roadmap#DB и аналитика|разделе «DB и аналитика»]] среди заданий без сохранённого условия.

## Что отсутствует в контракте

- полный SQL-запрос и схема;
- объёмы и распределение данных;
- execution plan, индексы и целевая метрика оптимизации.

Карточка не объединяется с похожими задачами, пока основной контракт и ожидаемый результат неизвестны.
