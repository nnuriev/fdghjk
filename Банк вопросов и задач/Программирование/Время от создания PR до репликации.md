# Время от создания PR до репликации

## TL;DR

В материалах Avito сохранилось только название «Время от создания PR до репликации». Этого недостаточно, чтобы восстановить алгоритм, SQL или ожидаемый результат без догадок.

## Наблюдаемая формулировка

Единственная подтверждённая формулировка: **«Время от создания PR до репликации»**. Она перечислена в [[Авито/roadmap#DB и аналитика|разделе «DB и аналитика»]] среди заданий без сохранённого условия.

## Что отсутствует в контракте

- расшифровка `PR`;
- события начала и окончания измерения;
- система репликации, clocks и ожидаемый результат.

Карточка не объединяется с похожими задачами, пока основной контракт и ожидаемый результат неизвестны.
