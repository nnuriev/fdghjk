# Экран статистики

## TL;DR

В материалах Avito сохранилось только название «Экран статистики». Функциональные требования, нагрузка, API и ожидаемая глубина проектирования отсутствуют, поэтому архитектурное решение не восстанавливается.

## Контекст и ментальная модель

Наблюдаемые варианты: **«Экран статистики»**. Они перечислены в [[Авито/roadmap#Вне выбранного scope|разделе «Вне выбранного scope»]].

Источник относит название к клиентской постановке вне выбранного Go-backend scope. Архитектуру экрана по одному названию восстановить нельзя.

## Требования

### Функциональные

Неизвестны. Нужно восстановить:

- пользователь и продуктовая цель экрана;
- набор метрик и источник данных;
- refresh, consistency, offline behavior и критерии оценки.

### Нефункциональные и SLO

Неизвестны: источник не сохраняет нагрузку, latency, availability, durability, consistency и resource limits.

### Вне scope

Неизвестно до восстановления исходного условия. Сам title-only пункт уже помечен источником как находящийся вне выбранного Go-backend scope.
