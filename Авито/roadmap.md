# Avito — roadmap для Go-backend интервью

## Как устроен маршрут

Маршрут идёт от наиболее вероятных задач к интеграционным: две рекомендованные алгоритмические задачи, Backend Platform Go, теоретический screening, четыре полных system-design кейса, затем редкие задачи. Условия и дубли исходников разобраны в [[Авито/Источники|источниках Avito]].

Решение конкретной задачи находится в `Авито/Решения`, а переиспользуемая ментальная модель — в канонических заметках vault. Сначала стоит проговорить контракт и инвариант своими словами, затем воспроизвести решение и проверить failure paths.

## Версионная область

Go-код решений проверяется на Go 1.26.5. Для loop variables отдельно показаны module language version `go 1.21` и семантика начиная с Go 1.22. Инфраструктурные заметки фиксируют Kubernetes 1.36.2, PostgreSQL 18.4 и PostGIS 3.6.4 по состоянию на 2026-07-18.
