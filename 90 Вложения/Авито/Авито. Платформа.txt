1. # ------------ Сборка сниппета ------------ #
Условие задачи

Мы делаем сервис, который собирает сниппет товара. Каждый сниппет состоит из отформатированного описания и стоимости товара в рублях.

Чтобы собрать сниппет нужно:
1. Получить описание из одного сервиса, а затем отформатированное его через prettify
2. Получить цену (в долларах) из другого сервиса, а затем перевести ее в рубли через pricetoRub
3. Вернуть готовый сниппет

- Асинхронность:
— Отличие между тредами, процессами и корутинами?
— Как работает веб-сервер?

2. # ------------ Сетевые запросы ------------ #
Условие задачи

Что выведет следующая программа и сколько она будет выполняться по времени?
package main

import (
    "fmt"
    "time"
)

const numRequests = 10000

var count int

func networkRequest() {
    time. Sleep(time.Millisecond) // Эмуляция сетевого запроса
    count++
}

func main() {
    for i := 0; i < numRequests; i++ {
        networkRequest()
    }

    fmt.println(count)
}
